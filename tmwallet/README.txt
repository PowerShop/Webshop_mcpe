สคริปที่ผมแจกนี้ผมได้ดัดแปลงมาจากสคริปเติมเงินเข้าเกมส์ GTA Online นะครับ (http://www.game-gta.com/board/index.php?topic=5701.0)

ชี้แจงก่อนนะครับ
- ผมยังไม่ได้ทดสอบนะครับ เพราะหาหนูทดลองไม่ได้


หลักการทำงาน
ระบบนี้เป็นระบบของ TMTopup.Com นะครับ ซึ่งเป็นระบบตัดบัตรเงินสดอัตโนมัติ ให้บริการฟรี !!
โดยเมื่อเรากรอกรหัสบัตรที่หน้าเว็บไซต์ของ TMTopup แล้วระบบจะตรวจเช็คบัตรว่าถูกต้องหรือไม
หากถูกต้อง เงินในบัตรจะถูกหักเข้าบัญชีเราที่สมัครกับทาง TMTopup เอาไว้นะครับ
แล้วให้ระบบ TMTopup ส่งข้อมูลมายังเว็บไซต์เราผ่านสคริปที่ผมแจกนี้
จากนั้นสคริปนี้จะทำการเพิ่ม Point หรือ ยอดเงินเข้าสู่ฐานข้อมูลของสมัครชิก


วิธีการติดตั้ง&ใช้งาน
1. ต้องเป็นสมาชิก http://www.tmtopup.com ก่อนนะครับ (สมัครฟรี ไม่มีค่าใช้จ่ายใดๆ)
การสมัครต้องใช้เลขบัตรประชาชนของจริงเท่านั้น

2. ดาวน์โหลดไฟล์ TMTopup API By Nst.zip ที่ http://www.mediafire.com/?gdgljruk2tirpoa

3. แตกไฟล์ TMTopup API By Nst.zip ที่ดาวน์โหลดมาจะได้ไฟล์ 3 ไฟล์ คือ AES.php, Rijndael.php, tmtopup_api_pp.php

4. เปิดไฟล์ tmtopup_api_pp.php ขึ้นมา (ใช้โปรแกรมอะไรเปิดนี้ก็...แล้วแต่ท่านเลย Notepad, EditPlus)

5. จัดการแก้ไขข้อมูลต่างๆ ให้ตรงกับเว็บไซต์เราดังนี้
# ------------------------------------- Config Begin ------------------------------------- #
// ------------------------------------------------------------------------------------------------
/* MySQL Config | Begin */
// Hostname ของ MySQL Server
$_CONFIG['mysql']['dbhost'] = 'localhost';

// Username ที่ใช้เชื่อมต่อ MySQL Server
$_CONFIG['mysql']['dbuser'] = 'username';

// Password ที่ใช้เชื่อมต่อ MySQL Server
$_CONFIG['mysql']['dbpw'] = 'password';

// ชื่อฐานข้อมูลที่เราจะเติม Point ให้
$_CONFIG['mysql']['dbname'] = 'gta';

// ชื่อตารางที่เราจะเติม Point ให้ ตัวอย่าง : member
$_CONFIG['mysql']['tbname'] = 'member';

// ชื่อ field ที่ใช้อ้าง Username
$_CONFIG['mysql']['field_username'] = 'username';

// ชื่อ field ที่ใช้ในการเก็บ Point จากการเติมเงิน
$_CONFIG['TMN']['point_field_name'] = 'point';
/* MySQL Config | End */
// ------------------------------------------------------------------------------------------------


// ------------------------------------------------------------------------------------------------
/* จำนวน Point ที่จะได้รับเมื่อเติมเงินในราคาต่างๆ | Begin */
$_CONFIG['TMN'][50]['point'] = 500;	// Point ที่ได้รับเมื่อเติมเงินราคา 50 บาท
$_CONFIG['TMN'][90]['point'] = 900;	// Point ที่ได้รับเมื่อเติมเงินราคา 90 บาท
$_CONFIG['TMN'][150]['point'] = 1500;	// Point ที่ได้รับเมื่อเติมเงินราคา 150 บาท
$_CONFIG['TMN'][300]['point'] = 3000;	// Point ที่ได้รับเมื่อเติมเงินราคา 300 บาท
$_CONFIG['TMN'][500]['point'] = 5000;	// Point ที่ได้รับเมื่อเติมเงินราคา 500 บาท
$_CONFIG['TMN'][1000]['point'] = 10000;	// Point ที่ได้รับเมื่อเติมเงินราคา 1,000 บาท
/* จำนวน Point ที่จะได้รับเมื่อเติมเงินในราคาต่างๆ | End */
// ------------------------------------------------------------------------------------------------


// กำหนด API Passkey สำหรับเชื่อมต่อกับระบบ TMTopup
define('API_PASSKEY', 'NSTPasskey01');

# -------------------------------------- Config End -------------------------------------- #



6. อัพโหลดไฟล์ PHP ทั้ง 3 ไฟล์ เข้า Web Server ของคุณ

7. Login เข้าเว็บ tmtopup.com เลือกเมนู "ตั้งค่าการเติมเงิน"

8. ตั้งค่า Ref1 , Ref2 , API URL และ API Passkey ดังนี้
- API URL ระบุ URL ของไฟล์ tmtopup_api_pp.php ที่อัพโหลดไว้ในข้อ 6 เช่น http://www.thaicreate.com/tmtopup/tmtopup_api.php
- API Passkey ระบุ API_PASSKEY ที่กำหนดไว้ในไฟล์ tmtopup_api_pp.php ในข้อที่ 3 เช่น NSTPasskey01
- Ref1 กำหนดให้ลูกค้าใส่ Username ของเว็บไซต์ หรืออย่างอื่นที่อ้างถึงตัวสมาชิกท่านที่เติมเงิน เวลาเงินเข้าจะใช้สิ่งนี้ในการอ้างอิงเติม Point
- Ref2 จะให้ลูกค้าใส่อะไรก็ได้ เช่น ชื่อเล่น หรือ Email หรือ เบอร์โทรศัพท์ติดต่อ

9. คลิ๊กเมนู "ภาพรวมบัญชี" เอาลิ้งค์เติมเงินไปให้ลูกค้าเติมได้ทันที หรือจะทำ iframe ไปแปะที่หน้าเว็บไซต์ท่านก็แล้วแต่ครับ

10. จบการติดตั้ง ทดสอบการทำงาน
